# Pic-Tune (피크튠)

> 사진의 분위기를 AI가 분석해, **시각적 감성을 청각적 플레이리스트로 즉시 변환**해주는 초개인화 서비스

## 프로젝트 소개

### 문제 정의 (Pain Point)

**1. 원하는 감성의 음악을 찾는 데 시간이 오래 걸림**
- 지금의 감정/장소/상황에 어울리는 음악을 찾고 싶지만, 직접 탐색하는 과정이 번거롭다
- 무드는 명확한데 3~4곡 정도만 떠오르고, 충분한 트랙을 빠르게 확보하기 어렵다

**2. 플레이리스트 제작 및 편집 과정의 번거로움**
- 음원 추출, 이미지 생성, 영상 편집 등 단계별로 각기 다른 사이트와 앱을 번갈아 사용해야 한다
- 곡 간의 전환(Crossfade), 재생 시간 조정 등 단순 작업에 과도한 시간이 소요된다

### 해결 방안 (Solution)

**1. 키워드 입력 대신, '사진 업로드'를 무드 입력으로 활용**
- 사진을 업로드하면 AI가 이미지 분위기를 분석해 무드를 정의하고, 음악을 자동 추천
- 사용자의 탐색 시간을 줄이고, 원하는 분위기에 더 빠르게 도달

**2. 영상에 바로 삽입 가능한 "하나의 음원 파일"로 제공**
- 생성된 플레이리스트를 하나의 음원 파일 형태로 제공해 영상 제작 시 바로 활용 가능

## 주요 기능 (Features)

### F1. 사진 업로드 기반 Mood 분석
사용자가 업로드한 이미지를 AI가 분석해 분위기 키워드를 추출합니다.

- Mood 키워드 3~6개 제공
- 키워드 추출 근거 설명 (신뢰성 확보)
- 예시: 청량한 / 잔잔한 / 도시적인 / 따뜻한 / 몽환적인 / 새벽감성

### F2. 플레이리스트 생성
추출된 무드를 기반으로 15곡을 추천하고, 감성 타이틀을 자동 생성합니다.

**타이틀 예시:**
- "비 오는 날 창가에서 듣는 회색빛 R&B"
- "초여름 저녁의 청량한 인디팝"

### F3. 추천 근거 제공 (Explainability)
사용자가 추천 결과를 납득할 수 있도록 근거를 제공합니다.

- AI가 도출한 감성 키워드 표시
- 사진의 어떤 요소(밝기/색감/풍경/인물/시간대)가 해당 무드인지 설명

### F4. Mood Slider (사용자 컨트롤)
사용자가 플레이리스트 무드를 미세 조정할 수 있는 6개의 슬라이더를 제공합니다.

| 축 | 범위 |
|---|---|
| 에너지 | 힙함 ↔ 잔잔함 |
| 템포 | 신나는 ↔ 차분한 |
| 온도 | 따뜻한 ↔ 차가운 |
| 명도 | 밝은 ↔ 어두운 |
| 분위기 | 몽환적인 ↔ 선명한 |
| 밀도 | 미니멀한 ↔ 풍성한 |

### F5. 다시 생성 (Regenerate)
결과가 마음에 들지 않으면 슬라이더 조절 후 "다시 만들기"가 가능합니다.

### F6. 커버 이미지 3종 생성
플레이리스트 커버 아트 3종을 제공합니다.

1. **원본 사진 버전**
2. **유튜브 썸네일 스타일 버전**
3. **일러스트 앨범 커버 스타일 버전**

## 시작하기 (Getting Started)

### Prerequisites
- Node.js 20 이상
- pnpm

### Installation

```bash
# 저장소 클론
git clone <repository-url>
cd kanana-forest

# Node 버전 설정 (nvm 사용 시)
nvm use

# 의존성 설치
cd frontend
pnpm install
```

### Development

```bash
# 개발 서버 실행
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

### Build

```bash
# 프로덕션 빌드
pnpm build

# 프로덕션 서버 실행
pnpm start
```

## 타겟 사용자 (Target Users)

### 핵심 타겟: 플레이리스트 유튜버

**페르소나 1: 지민 (26)**
- 분위기 있는 플레이리스트 콘텐츠를 빠르게 제작하고 싶음
- 무드는 선명한데 곡 탐색 시간이 오래 걸림
- 사진만으로 무드가 바로 잡히고, 제목/커버까지 한 번에 완성되길 원함

**페르소나 2: 서연 (24)**
- 채널 성장을 위해 제작 주기를 단축하고 싶음
- 음원 추출, 커버 보정, 영상 편집 등 2~3개 앱을 번갈아 사용하는 것이 비효율적
- 편집 시간을 줄여주는 영상 소스 형태의 결과물을 원함

## 경쟁 우위 (Competitive Advantage)

### vs. AI 챗봇 (ChatGPT, Gemini)

| 구분 | AI 챗봇 | Pic-Tune |
|------|---------|----------|
| 사용 방식 | 한 번 만들고 끝, 매번 프롬프트 입력 | 업로드 → 결과 → 조절 → 재생성 |
| 무드 조정 | 다시 말로 설명해야 함 | 무드 슬라이더로 즉시 컨트롤 |
| 결과물 | 노래 리스트만 제공 | 플레이리스트 + 타이틀 + 커버 3종 |
| 활용성 | 사용자가 따로 편집 필요 | 바로 업로드 가능한 형태 |

**핵심 차별점:**
- 대화가 아닌 **제작 도구(툴)**로 제품화
- 크리에이터가 원하는 **완성된 납품물** 제공
